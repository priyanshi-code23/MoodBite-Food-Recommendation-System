MoodBite 🍔

MoodBite is a mood-based food recommendation system built using Python, Streamlit, and PostgreSQL.

Features

- User Login
- Food Menu
- Add to Cart
- Wishlist
- Order Management
- Mood-Based Food Recommendations
- Estimated Delivery Time
- Order Confirmation Messages

Technologies Used

- Python
- Streamlit
- PostgreSQL
- psycopg2

Database Setup

For security reasons, database credentials are not included in this repository. Use "db_example.py" as a reference and rename it to "db.py" after adding your PostgreSQL credentials.

Run the Project

Install the dependencies from "requirements.txt" and run:

streamlit run app.py
